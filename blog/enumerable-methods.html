Using Enumerable Methods
Simple and basic usage
What is an Enumerable Method?:
Ruby utilizes methods that are built-in to their Arrays and Hashes. These methods are designed to perform actions on your Ruby Objects.
Start with the basics
A little code can really do a lot here!
[1, 2, 3].each { |number| puts number }
What else can they do?
Here we are using the .any? method. This method is used to get information on the Arrays's values. This code would tell us if any of the values are even or odd.
[1, 2, 3].any? { |number| number % 2 == 0 }
# => true
[1, 3, 5].any? { |number| number % 2 == 0 }
# => false.
Sometimes you need to know the index that you are currently on. You can use the .each_with_index method. Here we are creating a new hash, from an array, who's values are the arrays indexes.
hash = Hash.new
%w(cat dog wombat).each_with_index { |item, index|
   hash[item] = index
}
hash #=> {"cat"=>0, "dog"=>1, "wombat"=>2}
Closing Thoughts:
Enumerable methods are easy to learn, and fun to use. Ruby couldn't have made the learning process any easier. Knowing how these methods work, and being able to use them to get the desired results, its essential to being a programmer. Spend some time checking out the RubyDocs so you are familiar with them!

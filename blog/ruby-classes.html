Ruby Classes for newbees
A simple explanation
A stripped down class:
class MyNameClass
 def initialize name
  @name = name
 end
end
Explanation:
Above we have the most basic class definition. the initialize method will always be called, whenever we 'new up' a new object. What does this mean? Your code will look something like this. -->
my_name = MyNameClass.new('Chris')
 You have now 'new'd up' and object. We now have an object, we decided to call it 'my_name', this is an arbitrary variable name. This object has an instance variable, this again is arbitrary, we decided to call it '@name' The '@' sign signifies it is an instance variable. In our case, the value is 'Chris', because that is what we passed in when it was 'new'd up'. Lets say we need Another object. -->
my_friends_name = MyNameClass.new('Steve')
 Same as above, just another object. That's cool we have objects with instance variables whose values are different, but what do we do with them!!?? On to the next level.......

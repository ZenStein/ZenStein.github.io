Basics of SVG and CSS
can you find a more frustrating combination??
Scope:
I am going to assume you are a web developer who is interested in being able to embed svg's into your website, and while your at it, add some CSS styling to your svg. There is a ton of info out there, and this blog is not an attempt at adding to that info. This blog is to help a beginner get an overall understanding, and hopefully be able to avoid a few pitfalls.The following are some useful facts.
Good to know:
  • There are 2 ways to bring in an SVG. The simple basic way looks like this:

  • <img src="path/to/SOME_SVG.sgv" />

  • The other way, although a little more involved, allows for CSS manipulation. You embed the actual svg code into your html. You do this by opening your svg in a text editor, copying the code, and pasting it into you html. It will look something like this:
    <div>
    <svg> *** </svg>
    </div>
  • If you inline style an SVG, you cannot apply any CSS rules to it.
  • There are special rules, 'stroke' and 'fill' that are used to style only svg elements
